# 英単語ターゲット 習慣化サポーター (Target1900 Habit)

英単語（ターゲット1900など）の学習を継続するための、モバイルファーストな「進捗管理・習慣化Webアプリ」のMVPプロトタイプです。
Node.js などの複雑な環境構築を必要とせず、ブラウザだけで手軽かつ高速に動作します。

---

## 🚀 特徴

### 1. 黄金の学習サイクル
時間帯に合わせた4つの復習・インプットモードで、記憶の定着を最大化します。
*   **【朝】新規単語の閲覧**: 発音を聞きながら意味を素早く確認（オートプレイやタップ送り機能）。
*   **【昼】仕分けテスト**: Tinder風のカードUIを左右にスワイプし、「わかる」「わからない」を直感的に仕分けます。
*   **【夕方】過去の復習**: 習得済みの単語からシステムがランダムに抽出し、間隔反復（Spaced Repetition）で定着を確認します。
*   **【夜】今日の総復習**: 昼のテスト等で「学習中」に分類された単語を、すべて「習得済み」にするまで繰り返し出題するサドンデス形式です。

### 2. 進捗の可視化と習慣化（ダッシュボード）
*   **ストリーク表示**: 「〇日連続学習中！」を炎アイコンと共に表示し、モチベーションを維持します。
*   **全体・セクション別進捗**: 全セクションの習得度を直感的なプログレスバーや色分けバッジで可視化します。
*   **学習ヒートマップ**: GitHubのように、日々の学習量を緑の濃淡で記録します。

## 🛠️ 技術スタック

*   **フロントエンド**: HTML5, Vanilla JavaScript (ES Modules), CSS3
*   **スタイリング**: Tailwind CSS (CDN), Custom CSS (3D Flip / Tinder Swipe Physics)
*   **ライブラリ**: Papa Parse (CSV解析), Lucide Icons (アイコンフォント)
*   **API**: Web Speech API (音声読み上げ: TTS)
*   **データ管理**: LocalStorage (ブラウザ保存)

---

## 📂 ファイル構成

*   `index.html`: アプリケーションのメインUI構造（ダッシュボード、カードUI、CSVアップロード）
*   `app.js`: LocalStorage管理、CSV解析、Tinderスワイプ、音声読み上げ、学習ロジックなどのアプリ動作制御
*   `styles.css`: 3Dカードの反転、スワイプ時の回転など、Tailwind CSSを補完するアニメーション定義

## 📖 使い方

### 1. ローカルサーバーの起動
Pythonがインストールされている環境であれば、ターミナルで以下のコマンドを実行するだけで起動できます。

```powershell
# プロジェクトディレクトリ内で実行
python -m http.server 8000
```

起動後、ブラウザで [http://localhost:8000](http://localhost:8000) にアクセスします。

### 2. 単語データのインポート
1. 画面右上のアップロードアイコン（雲マーク）をクリックして「CSVインポート」画面に移動します。
2. 以下のフォーマットのCSVファイルをドラッグ＆ドロップ（またはファイル選択）してください。
    ```csv
    番号,単語,意味
    1,create,を創り出す；を引き起こす
    2,increase,増加する；を増やす
    3,improve,を向上させる；よくなる
    ```
    *※「セクション番号, 英単語, 日本語訳」の形式、および 1〜1900 の通し番号の形式（自動で100語単位でセクション分割されます）の両方に対応しています。*
3. テスト用に用意されている「サンプル単語をロードする」ボタンからデモデータですぐに試すことも可能です。
